## useState

- 第一參數為 state 的初始值


## useEffect

- 第一參數為 callback function
- 第二參數為 dependency array

實際上被 invoke 的時機點在 render 完之後，
並且與更新 DOM 物件是平行的，不會阻塞畫面進行重繪。

## useLayoutEffect

- 第一參數為 callback function
- 第二參數為 dependency array

實際上被 invoke 的時機點在 render 完之後，
並且早於更新 DOM 物件，同步執行，會阻塞畫面進行重繪。
